# Install
`npm install`

# Load soccer data
`npm load-data.js`

# Start app
`npm app.js`

# Public endpoints
* GET /upcominggames/:id
* GET /pastgames/:id
* GET /tournaments/:id
* GET /teams/:id

# Admin endpoints
* PUT|POST|DELETE /games/:id
* PUT|POST|DELETE /teams/:id
* PUT|POST|DELETE /tournaments/:id
